# wiki0

wiki0 is a local-first AI memory project.

## Shape

- Markdown wiki as canonical storage.
- SQLite index for fast search, backlinks, facts, and graph data.
- CLI and MCP interfaces for agents.
- SvelteKit web app for human browsing and review.

## Dogfooding

wiki0 is now being used to track its own product and architecture
decisions. See [[projects/building-wiki0]] for the active build log
and current stance.
